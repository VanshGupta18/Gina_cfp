# Contributing to GINA

## Developer Certificate of Origin (DCO)

This project expects commits that are compatible with the **Apache License 2.0** and the **Developer Certificate of Origin**, as required by the hackathon terms.

When you commit, use a sign-off line:

```bash
git commit -s -m "Your message"
```

This adds:

```text
Signed-off-by: Your Name <your.email@example.com>
```

Use **one consistent email address** for all commits and hackathon-related communication, per organizer rules.

**Repository visibility:** keep the GitHub repository **private** during the hackathon unless the organizers instruct otherwise.

## What the sign-off means

By signing off, you certify the [DCO 1.1](https://developercertificate.org/) statement: you have the right to submit the work, it is licensed appropriately, and you are not including confidential or third-party material you are not allowed to distribute.
